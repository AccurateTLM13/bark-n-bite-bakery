# Bark-n-Bite Bakery

## Prerequisites

- **Node.js** 18 or higher is recommended for running the development tools.

## Install dependencies

```bash
npm install
```

## Run the development server

```bash
npm run dev
```

This starts a Vite server and serves the project at `http://localhost:5173` by default.

## Build for production

```bash
npm run build
```

This creates an optimized production build in the `dist` folder.
