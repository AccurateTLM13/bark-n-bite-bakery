# Bark & Bite Bakery

A small demo site built with Vite and React.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Create a `.env` file based on `.env.example` and customize the values:
   ```bash
   cp .env.example .env
   ```
   The following variables control the contact information displayed in the footer:
   - `VITE_CONTACT_PHONE`
   - `VITE_CONTACT_EMAIL`
   - `VITE_CONTACT_LOCATION`
   - `VITE_PICKUP_HOURS`
   - `VITE_ORDER_CLOSE`

3. Start the development server:
   ```bash
   npm run dev
   ```

## Build

```bash
npm run build
```
